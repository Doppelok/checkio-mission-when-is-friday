Friday is an awesome day. The end of the week, when you are waiting for the weekends and interesting events. It's good to know how many days are left to it, does it?

Your task is to write a function which should to count this. The argument will be a date in string format look like - 'dd.mm.yyyy' where 'dd' - day, 'mm' - month and 'yyyy' - year.
For example, if that day is Thursday - the answer will be 1. If that day is Monday, the result is 4. And if that day is Friday - you should return 0.

Input: Date as a string.

Output: The amount of days left to the Friday.

Precondition:
1 <= year <= 3000
